jadice server :: Converter Client

This is a demo application for levigo® jadice server 5. You can either use it to trigger various conversion jobs or to monitor a remote instance of jadice server.

For the latest version of this client visit our github site. You are also welcome to fork the repository, make improvements and send us a pull request.

If you are interested in a evaluation version of levigo® jadice server 5 or want get get more information about it, contact us at jadice-support@levigo.de.

Usage

Conversion
Run misc. conversion jobs with predefinied workflow configurations
Server Log
Connect to the log of a remote instance of jadice server
JMX
Watch the performance of a remote instance of jadice server via JMX
Inspector
Inspect a workflow configuration
Options
Configuration of this application
About
Contains license infomation and the log of this application
